# Pricing Data Assumptions

SpendPilot uses heuristic pricing assumptions for optimization recommendations.

## Example Pricing

| Tool            | Plan     | Estimated Cost |
| --------------- | -------- | -------------- |
| Cursor Pro      | Pro      | $20/user       |
| Cursor Business | Business | $40/user       |
| ChatGPT Plus    | Plus     | $20/user       |
| ChatGPT Team    | Team     | $30/user       |
| Gemini Pro      | Pro      | $20/user       |
| Gemini Ultra    | Ultra    | $40/user       |

## Recommendation Logic

The recommendation engine evaluates:

- team size
- monthly spend
- subscription tier
- tooling redundancy

to estimate optimization opportunities.

These values are approximate and intended for demonstration purposes.
