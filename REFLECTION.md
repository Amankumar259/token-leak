# Reflection

Building SpendPilot taught me how production full-stack applications are structured beyond simple CRUD apps.

The biggest learning was understanding how frontend, backend, database infrastructure, deployment systems, and AI integrations connect together into a cohesive SaaS architecture.

I also learned the importance of:

- environment variable management
- CI/CD pipelines
- production-safe coding
- deployment debugging
- responsive UI design
- maintainable architecture

One of the most difficult parts was handling production deployment issues related to Supabase and Resend environment variables. Solving these issues improved my understanding of build systems and runtime environments.

If I continued developing the project, I would add:

- authentication
- advanced analytics
- real vendor pricing APIs
- billing integrations
- ML-based recommendation systems
- team collaboration workflows
